# AUTOMAÇÃO PARA COLETAR IMAGENS DE UM SITE

### CRIANDO MEU AMBIENTE VIRTUAL VENV

```
PS C:\dev\machine> git branch -M main
PS C:\dev\machine> python -m venv .env
PS C:\dev\machine> .\.env\Scripts\Activate.ps1
```
